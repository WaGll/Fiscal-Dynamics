
# Fiscal Dynamics

[English](./README.md)

## 项目简介
Fiscal-Dynamics 是一个基于中国省级经济数据的税收数据挖掘与计量经济学分析项目。

本项目围绕产业税收、居民消费支出以及居民可支配收入展开分析，
并结合统计学与机器学习方法完成区域税收分级研究。

## 项目内容
- 数据清洗与预处理
- 探索性数据分析（EDA）
- Pearson 相关系数分析
- 多元线性回归建模
- 异方差与自相关检验
- K-Means 聚类分析
- 地理空间可视化
